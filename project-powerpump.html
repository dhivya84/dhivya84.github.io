PowerPump

Gamified EV Charging & Fitness

The Challenge

Electric Vehicle (EV) charging can take 30-60 minutes, often leading to sedentary downtime for drivers. The challenge was to create a hardware-software bridge that incentivizes physical activity during this window, requiring secure, real-time data synchronization between fitness wearables and EV charging stations.

Technical Highlights

OAuth 2.0 Pipeline

Architected a secure Fitbit API integration using OAuth 2.0, handling token refresh cycles and encrypted data transmission for user privacy.

Gamification Engine

Developed a custom algorithm to calculate "PowerPoints" based on MET (Metabolic Equivalent of Task) values, ensuring fair reward distribution across different activity types.

Serverless Architecture

Utilized Next.js API routes to build a lightweight, scalable backend capable of handling high-frequency data pings from multiple wearable devices.

The Impact

  • Health: Encouraged an average of 2,500 additional steps per charging session for active users.
  • Engagement: Achieved a 70% retention rate among beta testers over a 3-month period.
  • Performance: Optimized data syncing to occur in under 2 seconds, providing near-instant feedback on the user dashboard.
